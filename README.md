# Open Data Schleswig-Holstein (opendata-schleswig-holstein-de)

Open Data Schleswig-Holstein is a state government open-data portal for Germany running **CKAN**, exposing a standardized catalog API over ~32,414 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/opendata-schleswig-holstein-de/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=opendata-schleswig-holstein-de-api-evangelist&utm_content=repo)
